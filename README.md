### Datasets

The HOTC2020 dataset is publicly available at https://www.hsitracking.com/hot2020/.
The IMEC25 dataset can be accessed at https://github.com/Chenlulu1993/HOMG.

### Requirements

Platform: MATLAB R2018a.
The pre-trained VGG-19 model used in this work is available at 
http://www.vlfeat.org/matconvnet/pretrained/.
The MatConvNet toolbox can be obtained from https://github.com/vlfeat/matconvnet.
